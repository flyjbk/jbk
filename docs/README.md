# 폴더 구조
┏assets┳css━styles.css : scss로 생성된 css 파일  
┃   ┣images : 이미지 파일 폴더  
┃   ┗scss┳abstracts : mixin, 색/사이즈 변수 등 파일  
┃      ┣base : reset.scss  
┃      ┣components  
┃      ┣pages : common.scss (전체 페이지 공통으로 들어가는)  
┃      ┗styles.scss : scss import하는 파일  
┗docs : html 파일
